# Co-host one-pager

You don't need Claude knowledge for any of this. Your job: AV, remote chat, and timing.

## Before 09:30

- [ ] Room screen + presenter laptop mirrored and readable from the back
- [ ] Video call live: room camera + room mic checked with one remote attendee
- [ ] Chat channel posted in the call and written on the room screen
- [ ] Pin in chat: link to claude.ai, the lab handouts, and `coffee_sales.csv`

## During labs (11:00–12:15 and 13:15–14:00)

- Watch the chat channel — this is your main job
- **Login/technical issue** → try once ("refresh, re-login"), if not solved in 2 min, post the person's name + issue so the presenter picks it up between desks
- **Content question** ("why did Claude do X?") → don't answer; queue it. Post to presenter at the next natural pause
- Every ~20 min, ask remote folks: "all following along? anyone stuck?"

## During lectures/demos

- Monitor chat for "can't see / can't hear" — fix AV immediately, interrupt if needed
- Collect questions in a list; presenter takes them at block ends, not mid-demo
