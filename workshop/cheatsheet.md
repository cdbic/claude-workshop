# Claude — Start Tonight Cheat Sheet

*Links current as of July 2026.*

## The ladder

1. **Free** — chat, artifacts, file analysis, web search, connectors, memory → https://claude.ai
2. **Pro (~$20/mo)** — adds Claude Code, Claude Cowork, Projects, Research, more models → https://claude.com/pricing
3. **Your AI brain** — skills + schedules + memory files that compound over time (what you saw this afternoon)

## Do tonight (free, 15 minutes)

- [ ] Redo Lab 1 Exercise 2 with your own idea — build an artifact you'd actually use
- [ ] Upload a spreadsheet from your real work and ask for three insights
- [ ] Connect one service (Drive/GitHub) and ask Claude something about your own stuff
- [ ] Tell Claude one durable preference and watch it stick across conversations

## When you're ready for the agent (paid)

- **Claude Code** — the terminal agent from Demo 1. Get Pro, then: `npm install -g @anthropic-ai/claude-code`, run `claude` in any project folder
- **Claude Cowork** — the desktop agent from Demo 3; included with Pro, in the Claude desktop app
- Start your "AI brain" small: one `CLAUDE.md` file in one repo describing how you work — everything else grows from there

## Prompting crib

- Say **who it's for, what format, how long** — specificity beats magic words
- Show **one example** of what you want — the pattern comes back
- Treat answer #1 as a **draft** — iterate, don't restart
- Facts that matter → ask for **sources** (web search) and check one
